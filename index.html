Agreement

By using the discord server and or bot, you agree to the usage of your publicly avaliable discord account data for issues, abuse, punishment, and blacklists.

Upon joining, you must type !agree to gain access to the server. If, at any time you disagree to the rules, you may leave the server.

Join us on Discord

Commands

!help

Display a list of commands and required permissions.

Usage:

!help

!uptime

View the bot's current uptime.

Usage:

!uptime

!dev

View the bot's contributors

Usage:

!dev

!suggest

Make a suggestion

Usage:

!suggest [Your Suggestion]

Sub-commands for !suggest

Permission

These commands require the ADMINISTRATOR permission or higher.

!deny:

!deny [Suggestion ID] [Reason]

!accept:

!accept [Suggestion ID] [Reason]

!consider:

!consider [Suggestion ID] [Reason]

!delete:

!delete [Suggestion ID]

!slow

Permission

This command requires the MANAGE_CHANNEL permission or higher.

Set current channel's slowmode in seconds.

Usage:

!slow [Number]

!kick

Permission

This command requires the KICK_MEMBERS permission or higher.

Kick a member from the server

Usage:

!kick @Member [Reason]

!ban

Permission

This command requires the BAN_MEMBERS permission or higher.

Ban a member from the server

Usage:

!ban @Member [Reason]
